## Readiness Checklist Reference

- 제목과 메타 설명이 검색 의도와 맞는지 점검
- 첫 문단에서 왜 지금 읽어야 하는지 말했는지 점검
- 핵심 숫자와 날짜를 발행 직전 다시 확인
- 관련 허브/evergreen 글 내부링크 최소 2개 연결
- 광고를 넣더라도 첫 화면 과밀 배치 피하기
- 면책문구와 운영 주체 표시 확인
