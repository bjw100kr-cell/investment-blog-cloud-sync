# 홈/메뉴 구조 제안

- 사이트명: 돈의흐름노트
- 태그라인: 주식, 코인, 거시경제를 쉽게 풀어쓰는 투자 해설 블로그

## 상단 메뉴

- 거시경제
- 코인
- 세계 흐름·섹터
- About
- 운영 원칙

## 홈 화면 구성

- 상단: 오늘의 핵심 글 1개
- 중단: 최신 글 3~5개
- 하단: 카테고리 허브 3개
- 사이드/하단: 주간 회고, 인기 해설 글

## 수익화 메모

- 초보 투자자용 설명형 글은 검색 유입과 광고 친화성이 높다
- 허브 글은 체류 시간을 늘리고 내부링크 구조를 단단하게 만든다
- 속보형 글은 초기 유입, 설명형 허브 글은 장기 유입을 담당한다
